2.运算符

1.如果像定义了 char b = ‘c’ int a = b; 那么他就会发生自动类型转换 ，把char类型的转换成int类型的

2.优先级 先是String 再是int 比如当 String类型的变量加上int 的变量时候 那时候的+号变成了连接符

自增++ 自减– 比如 ++num num++

当自增单独使用的时候 num++和++num 没有任何区别

A.当混合使用的时候 那区别很大 ，比如++num,那么num立刻加上+1，然后拿着结果进行使用 【先加后用】

B.如果是num++,那么首先使用变量本身的数值，然后再让变量加1.【先用后加】

注意：常量不可以用自增或者自减 ，因为常量本身是不可变的

逻辑运算符

逻辑与（并且） && 逻辑与 的条件是 ：两边为真则为真,否则为假 例如：true && true 否则为false

逻辑或（或者） || 逻辑或的条件是：一边为真则为真 例如：true || true 真 , true || false 真

false || false 则假

逻辑非（取反） ！ 逻辑非的条件是：是true 为false 是真则为假

注意：逻辑与和逻辑或 具有短路的效果：如果根据左边已经可以判断得到了最终的结果，那么右边的话就将不在执行，因为第一个条件没有满足。

先条件判断，如果条件判断为 true 则把表达式A的赋值为变量，如果为false 则把表达式B赋值给变量

方法的重载

当对于功能类相同的或者类似的来说，因为参数列表不一样，却需要记住那么多不同的方法名称，太麻烦。

方法的重载（Overload）:多个方法名称相同，但是列表不一样。

好处：只需要记住唯一一个方法名称，就可以实现类似的多个功能。

方法重载的注意事项

方法重载与下列因素相关：

（1）参数个数不同

（2）参数类型不同

（3）参数的多类型顺序不同

方法重载与下列因数无关：

（1）与参数的名称无关

（2）与方法的返回值类型无关

数组

数组的概念：是一种容器，可以同时存放多个数据值

数组的特点：

1.数组是一种引用数据类型

2.数组当中的多个数据，类型必须全部统一

3.数组的长度在程序运行期间不可改变

数组的初始化：在内存当中创建一个数组，并且向其中赋予一些默认值。

数组有常见的两种初始化：

1.动态初始化（指定长度）

2.静态初始化（指定内容）

动态初始化数组的格式

数据类型[] 数组名称 = new 数据类[数组长度];

动态初始化数组的格式

标准

数据类型[] 数组名称 = new 数据类[]{元素1,元素1,元素1,元素1,元素n}

简易

数据类型[] 数组名称={元素1,元素1,元素1,元素1,元素n}
注意：

直接打印数组名称，得到的是数组对应的，内存地址哈希值

如果想访问数组元素就要索引值 格式：数组名称[索引值]

例如：System.out.println(数组名称[0])

0 表示下标 也是索引值

使用动态初始化数组的时候，其中的元素将会自动拥有一个默认值。规则如下：

如果是整数类型，那么默认为0

如果是浮点类型，那么默认为0.0

如果是字符型，那么默认为’\u0000’

如果是布尔类型，那么默认为false

如果是引用类型，那么默认为null
