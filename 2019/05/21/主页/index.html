当我们想上传gitee的时候

（1）第一步：如果没有设置本地仓库的时候 就需要先设置仓库 用 git clone https://gitee.com/liyongtie/neoDome01.gie 去设置

git clone（是命令） https://gitee.com（是gitee） liyongtie(是gitee的用户名) neoDome01.git（码云上创建一个仓库的名称）

附图（1）：

在那个盘符下面打开git bash 然后就创建了名为neoDome01的文件夹;

附图（2）：

（2）第二步:然后在把你需要的项目添加到名为neoDemo01的本地仓库中（也就是neoDome01中）然后执行 git add. 是添加（所有neoDome01中的项目） 或者也可以用 gitee add 后面跟的是要添加的项目名 比如（测试文件.txt）

（3）第三步：用git commit -m “（描述新添加的文件内容）.也就是注释”

（4）第四步：用git push -u origin master 意思就是将本地仓库（neoDome01）推送到远程仓库
